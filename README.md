# beesag

- bee genome resource
- single source that brings all bee genomics together along with the machine and deep learning. 
- first graphql based bee genomics. 
- Neural autoencoder and neural classification for bees infection and swarming rates.
- Machine learning for the toxic analysis and expression submergence.
